register(symbol, parent_type=:html, string=nil, extension_synonyms = [])

add new mime types for mobile devices: the second parameter indicates that a type is a "subtype" of another one. This way you can create mime types for specific devices, browser versions or firmware patchlevels. If no view template for a specific type is found, a more general view template is chosen.
